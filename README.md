# Skills

A collection of custom skills for [Claude Code](https://claude.ai/claude-code).

## Available Skills

| Skill | Description |
|---|---|
| [rocky-speak](./rocky-speak/) | Talk like Rocky, the Eridian alien from Project Hail Mary |

## Installation

Install all skills:

```bash
npx skills add BharathxD/skills
```

Install a specific skill:

```bash
npx skills add BharathxD/skills -s rocky-speak
```

You can also list available skills before installing:

```bash
npx skills add BharathxD/skills -l
```
