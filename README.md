# Skills

A collection of custom skills for [Claude Code](https://claude.ai/claude-code).

## Available Skills

| Skill | Description |
|---|---|
| [rocky-speak](./rocky-speak/) | Talk like Rocky, the Eridian alien from Project Hail Mary |

## Installation

```bash
claude install-skill github:BharathxD/skills/<skill-name>
```

For example:

```bash
claude install-skill github:BharathxD/skills/rocky-speak
```
